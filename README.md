# 一次函数 y = kx + b 可视化演示

## 项目简介

这是一个用于教学和学习的一次函数可视化演示网页，通过交互式界面帮助学生理解一次函数的原理、斜率和截距的几何意义。

## 功能特性

### 1. 📊 实时可视化
- 动态绘制一次函数图像
- 清晰的二维坐标系，带网格和刻度
- 实时响应参数变化

### 2. 🎛️ 交互控制
- **斜率 k 滑块**：范围 [-5, 5]，步长 1
- **截距 b 滑块**：范围 [-5, 5]，步长 1
- **重置参数**：一键恢复默认值 (k=1, b=0)
- **随机函数**：随机生成新的函数参数

### 3. 📐 几何标注
- **截距点标注**：在 y 轴上清晰标记 (0, b) 点，并用虚线连接
- **斜率示意图**：通过直角三角形展示 Δy/Δx 的含义
- **动点坐标**：鼠标移到直线上可查看任意点的坐标

### 4. 📖 原理说明
- 动态解释当前 k 和 b 对图像的影响
- 详细说明斜率和截距的数学定义
- 实时显示当前函数表达式

## 使用方法

1. **调整参数**
   - 拖动滑块改变 k 和 b 的值
   - 观察直线如何随参数变化

2. **探索性质**
   - 尝试不同的 k 值，观察直线倾斜方向和陡峭程度
   - 改变 b 值，观察直线在 y 轴上的移动

3. **查看坐标**
   - 将鼠标移到直线上，查看该点的坐标
   - 观察坐标如何满足函数关系式

4. **快速操作**
   - 点击"重置参数"回到初始状态
   - 点击"随机函数"快速生成新示例

## 技术实现

- **纯前端实现**：HTML + CSS + JavaScript
- **Canvas 绘图**：使用原生 Canvas API 绘制坐标系和函数图像
- **响应式设计**：适配不同屏幕尺寸

## 教育应用

适用于：
- 初中数学一次函数教学
- 学生自主学习和复习
- 数学概念的直观演示
- 在线教学和课堂演示
