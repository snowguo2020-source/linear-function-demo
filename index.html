一次函数 y = kx + b 可视化

一次函数是指形如 y = kx + b（其中 k ≠ 0）的函数，其图像是一条直线。

斜率 k：表示直线的倾斜程度，k > 0 时直线向右上方倾斜，k < 0 时向右下方倾斜。|k| 越大，直线越陡峭。

截距 b：表示直线与 y 轴的交点纵坐标，即当 x = 0 时，y = b。

实际应用：一次函数广泛应用于日常生活，如计算出租车费用（起步价 + 单价×里程）、温度转换（摄氏度与华氏度）、速度与路程的关系等。

当前函数表达式

y = 1x + 0

原理说明

📐 斜率 k 的含义

斜率 k = Δy / Δx，表示 x 每增加 1 个单位，y 增加 k 个单位。

  • k > 0：直线向右上方倾斜（递增）
  • k < 0：直线向右下方倾斜（递减）
  • |k| 越大，直线越陡峭

📍 截距 b 的含义

截距 b 是直线与 y 轴的交点，坐标为 (0, b)。

  • b > 0：直线与 y 轴交于正半轴
  • b < 0：直线与 y 轴交于负半轴
  • b = 0：直线经过原点
